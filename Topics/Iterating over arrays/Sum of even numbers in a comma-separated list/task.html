Sum of even numbers in a comma-separated list

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a Java program that takes as input a list of integers separated by commas, and calculates and prints the sum of all even numbers in the list. The list may contain both positive and negative integers.
Sample Input:
2,4,6,8,10

Sample Output:
30


Sample Input:
1,2,3,4,5

Sample Output:
6



Memory limit: 256 MB
Time limit: 5 seconds
