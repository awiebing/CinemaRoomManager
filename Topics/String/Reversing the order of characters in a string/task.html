Reversing the order of characters in a string

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a program that takes a string as input and prints a new string where the order of characters has been reversed. For example, if the input string is 'hello', the output will be 'olleh'.
Sample Input:
hello

Sample Output:
olleh


Sample Input:
world

Sample Output:
dlrow



Memory limit: 256 MB
Time limit: 5 seconds
